# Acerca de

El sistema de inventario permite registrar productos, consultar existencias y generar reportes básicos.

## Funciones principales

| Función | Descripción |
|---------|-------------|
| Registro de productos | Permite agregar nuevos productos al sistema |
| Consulta de stock | Muestra existencias disponibles |
| Reportes | Genera reportes básicos de inventario |
