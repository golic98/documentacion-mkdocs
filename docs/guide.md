# Guía

!!! tip
    Usa `mkdocs serve` para revisar los cambios localmente antes de publicar.

## Ejemplo de instalación

=== "Windows"
    ```bash
    python -m venv .venv
    .venv\Scripts\activate
    pip install mkdocs mkdocs-material pymdown-extensions
    ```

=== "Linux/macOS"
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    pip install mkdocs mkdocs-material pymdown-extensions
    ```
